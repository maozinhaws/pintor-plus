# Pintor Plus - Projeto de Correção e Melhoria

## Visão Geral
O Pintor Plus é um Progressive Web App (PWA) para pintores profissionais criarem orçamentos, gerenciarem clientes e sincronizarem com o Google Drive. Este projeto visa corrigir bugs críticos e melhorar a experiência do usuário.

## Problemas Identificados
1. **Erros na geração de PDF** - Problemas com recibos e orçamentos em PDF
2. **Problemas com sincronização** - Falhas na sincronização com Google Drive
3. **Bug de interface** - Teclado iniciando sobre o modal com opções de nomes para o item. O teclado deve ficar escondido até ser acionado para digitação.

## Impacto
Esses problemas estão impedindo totalmente o uso do app, especialmente as tarefas essenciais de criação de orçamentos, sincronização e geração de PDF.

## Funções Críticas
- Criação de orçamentos (adicionar itens, calcular valores, definir serviços)
- Sincronização com Google Drive (salvar e recuperar dados entre dispositivos)
- Geração de PDF (criar recibos e orçamentos profissionais)
- Cadastro de clientes (gerenciar informações dos clientes)
- Redução de burocracia e alto rendimento
- Orçamento rápido
- Acessibilidade para pessoas com baixa qualidade de visão e para pessoas leigas em tecnologia
- Diversificação do modo de preencher os dados (digitação, manuscrito ou voz)

## Recursos Técnicos
- O projeto usa JavaScript vanilla (sem frameworks)
- Dependência das APIs do Google para autenticação e armazenamento
- Deve funcionar offline com sincronização quando houver conexão
- É um Progressive Web App (PWA)

## Objetivos de Melhoria
1. Corrigir os bugs críticos que impedem o uso do app
2. Melhorar a experiência do usuário
3. Aumentar a acessibilidade
4. Otimizar o desempenho e a confiabilidade
5. Tornar o preenchimento de dados mais flexível
